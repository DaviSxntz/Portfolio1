Sistema Web Educação & Empregabilidade

SchoolFlow – Estágios & Aprendizagem

Sistema desenvolvido para facilitar o trabalho de articuladores entre escolas, empresas e alunos, reduzindo papelada e atrasos no processo de estágio e programas de aprendizagem.

  • Cadastro e gestão de alunos com edição de dados.
  • Geração automática de documentos: contrato, declaração de matrícula e ficha do aluno.
  • Cadastro de empresas parceiras e lançamento de vagas.
  • Dashboard com gráficos e tabelas mostrando a situação geral.
  • Contato rápido com o aluno via WhatsApp.

Tecnologias: HTML · CSS · JavaScript · PHP · MySQL
